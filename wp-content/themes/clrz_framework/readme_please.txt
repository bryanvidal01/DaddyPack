# Readme Please #

Bonjour, et merci d'utiliser Skeletorz.
Voici la liste des procédures à mettre
en place à chaque installation.

1. Configuration des fichiers
2. A Savoir


## Configuration des fichiers ##

- Il est nécessaire de renommer le dossier clrz_theme_default
vers un nom correspondant au nom court du projet.
( Généralement le nom du repository )
- Il est nécessaire de configurer le theme
avec des informations correspondant au projet.
Pour ce faire, modifiez l'entête du fichier /style.css en conséquence.

## A Savoir ##

- Les fichiers .css dans le sous dossiers css/ sont inclus dans l'ordre alphabétique.
- Le style CSS produit un fichier cache style-min.css ( dans le htdocs/www )
qui devra être mis en production à la place du style.php
